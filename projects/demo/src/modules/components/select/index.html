Use all powers of Textfield : put any number of Icons and Tooltips inside (and control their order and color), modify the size of the textbox and etc. Explore Textfield documentation page for more customization options. You can use Select with virtual scrolling from @angular/cdk/scrolling . Put <select tuiSelect/> (instead of <input tuiSelect/> ) to enable native browser picker.

This example demonstrates how enable this feature for mobile devices only.
